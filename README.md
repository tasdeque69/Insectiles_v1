## About The Game

**Insectiles** is a mesmerizing psychedelic tile-matching game that combines addictive gameplay with stunning visual effects and synthesized psytrance music. Tap insects in the correct order, build combos, and trigger fever mode for an explosive audiovisual experience.

### Features

- 🎮 **Addictive Gameplay** - Match insects in the correct order
- ✨ **Psychedelic Visuals** - Kaleidoscope, mandala, and trippy effects
- 🎵 **Synthesized Music** - Dynamic psytrance that responds to your play
- 🔥 **Fever Mode** - Double speed and explosive effects
- 🏆 **Leaderboards** - Compete with players worldwide
- 📱 **Cross-Platform** - Play on web, iOS, and Android

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/insectiles.git
cd insectiles

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
