## 1. Brand Overview

### 1.1 Mission Statement

**Insectiles** brings the psychedelic gaming experience to the mainstream, combining addictive gameplay with mind-bending visuals and immersive synthesized music.

### 1.2 Brand Values

| Value | Description |
|-------|-------------|
| **Psychedelic** | Bold, colorful, mind-expanding experiences |
| **Addictive** | Simple yet deep gameplay that keeps players coming back |
| **Immersive** | Audio-visual synergy that transports players |
| **Accessible** | Easy to learn, hard to master |
| **Premium** | Quality feel without the premium price tag |

### 1.3 Brand Personality

- **Playful** - Fun, energetic, exciting
- **Bold** - Not afraid to be different
- **Hypnotic** - Drawing players into the experience
- **Nostalgic** - Reminiscent of classic arcade games

## 2. Visual Identity

### 2.1 Logo

#### Primary Logo
- Full-color logo for light backgrounds
- White logo for dark backgrounds
- Minimum size: 32px height

#### Logo Clear Space
- Minimum clear space: 10% of logo height
- No other elements should encroach on this space

#### Logo Don'ts
- Don't stretch or distort the logo
- Don't change colors outside of brand palette
- Don't add effects (shadows, gradients) unless specified
- Don't place on busy backgrounds

### 2.3 Typography

#### Primary Font: **Orbitron**
- **Usage**: Headlines, titles, logo
- **Weights**: 400, 700, 900

#### Secondary Font: **Exo 2**
- **Usage**: Body text, UI elements
- **Weights**: 300, 400, 600

#### Display Font: **Press Start 2P**
- **Usage**: Score display, retro elements
- **Size**: Use sparingly for impact

#### Type Scale
| Element | Font | Size | Weight |
|---------|------|------|--------|
| H1 | Orbitron | 48px | 900 |
| H2 | Orbitron | 36px | 700 |
| H3 | Orbitron | 24px | 700 |
| Body | Exo 2 | 16px | 400 |
| Caption | Exo 2 | 14px | 300 |
| Score | Press Start 2P | 32px | 400 |

### 2.4 Iconography

#### Style
- Rounded corners (4px radius)
- 2px stroke weight
- Solid fills with subtle gradients

#### Icon Set
- Use Lucide icons as base
- Customize colors to match brand palette
- Maintain consistent 24px grid

### 2.5 Photography & Imagery

#### Style Guidelines
- High contrast, vibrant colors
- Psychedelic patterns and effects
- insects and nature elements
- Retro-futuristic aesthetic

#### Don't Use
- Generic stock photos
- Dull or washed-out images
- Realistic insect photography (use stylized)

## 3. Voice & Tone

### 3.1 Brand Voice

**Primary Tone**: Energetic and exciting  
**Secondary Tone**: Welcoming and accessible

### 3.2 Voice Characteristics

| Characteristic | Do | Don't |
|----------------|-----|-------|
| **Energetic** | Use exclamation marks, exciting words | Be dull or passive |
| **Playful** | Use fun language, emojis | Be overly serious |
| **Clear** | Explain features simply | Use jargon |
| **Confident** | Be bold in claims | Make unrealistic promises |

### 3.3 Sample Copy

| Context | Copy |
|---------|------|
| Game Title | **INSECTILES** - Psychedelic Tile Matching |
| Tagline | Tap. Match. Trip. |
| CTA | Start Playing |
| Feature | 🔥 Fever Mode - Double the speed, double the fun! |
| Score | Score: 1,000,000 |
