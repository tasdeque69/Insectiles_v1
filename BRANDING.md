## 🎯 Brand Overview

### Brand Name
**Primary:** Pinik Pipra  
**Tagline:** "Get high with the ants"  
**Pronunciation:** PIN-ik PIP-ra

### Brand Personality
- **Playful** - Fun, casual, approachable
- **Psychedelic** - Trippy visuals, mind-bending
- **Addictive** - Easy to play, hard to put down
- **Indie** - Unique, hand-crafted feel

### Target Audience
- **Primary:** Casual gamers (18-35)
- **Secondary:** Mobile gamers, puzzle enthusiasts
- **Tertiary:** Retro game enthusiasts

## 🎨 Visual Identity

### Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| **Electric Purple** | #8B5CF6 | Primary brand color |
| **Hot Pink** | #EC4899 | Accent, fever mode |
| **Cyan Glow** | #06B6D4 | Secondary accent |
| **Deep Black** | #0A0A0F | Background |
| **Pure White** | #FFFFFF | Text, highlights |
| **Lime Green** | #84CC16 | Success, score |

### Color Usage Rules
- ✅ Use Electric Purple for primary buttons
- ✅ Use Hot Pink for fever mode/celebrations
- ✅ Use Deep Black for backgrounds
- ✅ Use White for text on dark backgrounds
- ❌ Don't mix more than 3 colors in one element

---

## 🔤 Typography

### Primary Font
**Font Family:** System fonts (no external dependencies)
- iOS: SF Pro Display
- Android: Roboto
- Web: system-ui, sans-serif

### Font Weights
| Weight | Usage |
|--------|-------|
| 900 (Black) | Logo, headlines |
| 700 (Bold) | Buttons, CTAs |
| 500 (Medium) | Subheadlines |
| 400 (Regular) | Body text |

## 📐 Logo Usage

### Logo Elements
1. **Pink Ant Mascot** - The main character
2. **Text** - "PINIK PIPRA"

### Logo Variations
- **Full Logo** - Ant + Text (use for headers)
- **Icon Only** - Just the ant (use for app icon, favicon)
- **Text Only** - Just the text (use for small spaces)

### Clear Space
Always maintain **minimum 16px** clear space around logo.
